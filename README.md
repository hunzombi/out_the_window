# Out the Window

A simple text-based Python game made for a school project.

Throw your school supplies out the window and rack up points. Higher floors mean more points, and landing in the trash can is worth a big bonus!

## Requirements

- Python 3.8+

## How to play

```bash
python game.py
```

## Running the tests

```bash
python -m pytest tests/ -v
```
